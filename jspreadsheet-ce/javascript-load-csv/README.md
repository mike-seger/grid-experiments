# howto


## Spring Boot application (Server)

```
./gradlew bootRun
```

### Access Swagger-UI
http://localhost:8080/swagger-ui/index.html

## JS application (Client)

The root is in:
src/main/resources/static/

http://localhost:8080/ui/index.html

### Start local http server
```
python3 -m http.server 8000
```

### Access application
http://localhost:8000/
